# Manage members

As a catalogue manager with admin access, your primary focus is configuring [harvest sources](../add-harvest-sources). However, your admin role also grants you the capability to manage organisation members for exceptional cases—such as when a team member leaves or needs adjusted permissions.

## Organisation roles

CKAN provides three organisation-level roles, each with distinct permissions:

| Role       | Description                                                                                                |
| ---------- | ---------------------------------------------------------------------------------------------------------- |
| **Admin**  | Full control over the organisation, datasets, and members. Can configure harvest sources and manage roles. |
| **Editor** | Can create, edit, and delete datasets within the organisation.                                             |
| **Member** | Can view private datasets in the organisation.                                                             |


## Manage members

For situations where you need to add, modify, or remove organisation members, refer to the [CKAN organisation management documentation](https://docs.ckan.org/en/2.11/user-guide.html#managing-an-organization)<sup>↗</sup>.
