# Integrate external APIs

Integrate third-party services with REST patterns and authentication

This section covers integrating with external APIs and third-party services.

The GDI platform integrates with several external systems. This section provides patterns and examples for adding new integrations.

## Integration patterns

- REST client configuration in Quarkus
- Authentication (API keys, OAuth)
- Error handling and retries
- Rate limiting
- Response mapping
