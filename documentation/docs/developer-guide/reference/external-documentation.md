# External documentation

Access links to external documentation, standards, and resources relevant to the GDI User Portal.

## Standards and specifications

### DCAT-AP

- [DCAT-AP 3.0 Specification](https://semiceu.github.io/DCAT-AP/releases/3.0.0/) - Data Catalogue Application Profile for data portals in Europe

### FAIR Data Points

- [FAIR Data Point Specification](https://specs.fairdatapoint.org/) - Specification for FAIR Data Points
- [FAIR Data Point Repository](https://github.com/FAIRDataTeam/FAIRDataPoint) - Reference implementation

### RDF and Linked Data

- [RDF 1.1 Primer](https://www.w3.org/TR/rdf11-primer/) - Introduction to RDF
- [SPARQL 1.1 Query Language](https://www.w3.org/TR/sparql11-query/) - Query language for RDF
- [Turtle](https://www.w3.org/TR/turtle/) - Terse RDF Triple Language
- [JSON-LD](https://www.w3.org/TR/json-ld11/) - JSON for Linking Data

### SHACL

- [SHACL Specification](https://www.w3.org/TR/shacl/) - Shapes Constraint Language for validating RDF graphs

## Authentication and authorisation

### Keycloak

- [Keycloak Documentation](https://www.keycloak.org/documentation) - Identity and access management

### LS-AAI

- [Life Sciences AAI](https://lifescience-ri.eu/ls-login/) - Authentication and Authorisation Infrastructure for Life Sciences

### OpenID Connect

- [OpenID Connect Specification](https://openid.net/connect/) - Identity layer on top of OAuth 2.0
