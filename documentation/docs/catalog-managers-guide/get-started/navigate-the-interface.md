# Navigate the catalogue

The Catalogue Manager provides a straightforward interface for managing your genomic data catalogue. Access all core functions through the main navigation menu.

## Main navigation

The top navigation menu organises catalogue management into key areas:

<figure>
    <img src="/gdi-userportal-frontend/img/catalogue-managers-guide/get-started/navigation-menu.png" alt="Screenshot of the main navigation menu" width="900" />
    <figcaption></figcaption>
</figure>

    - **Datasets:** Manage all dataset records in your catalogue
    - **Organisations:** View and manage organisations and their datasets
    - **Groups:** Browse and manage groups used to organise datasets
    - **Harvest Sources:** Access and manage sources that automatically import datasets
    - **About:** View information about the catalogue

## Page layout

Each section uses a consistent layout: filters or contextual information appear on the left, while the main content displays on the right. Use the search bar to quickly find specific items by name or keyword.

## User controls

Access your personal settings and dashboard from the controls in the top-right corner:

<figure>
    <img src="/gdi-userportal-frontend/img/catalogue-managers-guide/get-started/user-controls.png" alt="Screenshot of the user controls in the top-right corner" width="350" />
    <figcaption></figcaption>
</figure>
- **System Admin Settings:** Configure system-wide settings (available to administrators only)
- **Profile:** Access your profile settings and preferences
- **Dashboard:** View your datasets, organisations, and groups
- **Login/Logout:** Sign in or out of your account
