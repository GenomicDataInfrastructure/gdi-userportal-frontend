# Manage group members

Groups support collaborative management through member roles. You can add members to groups and assign them specific roles that determine their level of access and capabilities.

## Group member roles

Each member in a group has one of the following roles:

- **Admin:** Can edit group information, as well as manage group members
- **Member:** Can add/remove datasets from groups
