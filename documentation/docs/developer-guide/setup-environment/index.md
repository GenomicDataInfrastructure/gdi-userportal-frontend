# Set up your development environment

Set up your development environment to start contributing to the GDI User Portal and its components. Choose your steps based on your role:

**Frontend developers**
1. [Install locally](./install-locally.md): Set up Node.js development environment
2. [Install with Docker](./install-with-docker.md): Run complete stack in containers

**Backend/Extension developers**
1. [Install locally](./install-locally.md): Set up CKAN, DDS, AMS development
2. [Configure authentication](./configure-authentication.md): Set up Keycloak and LS-AAI
3. [Troubleshooting setup](./troubleshooting-setup.md): Resolve common issues

**Platform operators**
1. [Install with Docker](./install-with-docker.md): Deploy full stack
2. [Configure authentication](./configure-authentication.md): Set up identity providers
3. Review [deployment guides](../deploy-production/index.md): for production environments
