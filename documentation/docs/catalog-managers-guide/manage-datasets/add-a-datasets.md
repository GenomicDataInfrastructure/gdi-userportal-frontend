# Add a dataset

Create a dataset record to make your genomic data resource discoverable in the GDI catalogue. If you're new to adding datasets, see: [What is a dataset?](what-is-a-dataset.md)

To add a dataset:

1. Go to **Datasets** in the main menu.

2. Select **Add Dataset**.

3. Fill out the form. Required fields are marked with an asterisk (*). <!-- DCAT link / dictionary for users to understand metadata terms -->
