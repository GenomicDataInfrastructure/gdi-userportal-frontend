# Manage datasets manually

Add and manage individual datasets directly in your catalogue when automated harvesting isn't suitable for your use case.

Manage datasets manually for:

- **One-off datasets**: Single datasets that don't require ongoing synchronisation
- **Test datasets**: Creating sample data for testing or demonstration purposes
- **Internal datasets**: Organisation-specific data not available from external sources
- **Special cases**: Datasets requiring custom metadata or unique handling

:::tip Use harvesting for automation

Manual dataset management is for exceptions. For regular updates, bulk imports, or external sources, use automated harvesting to reduce effort and keep your catalogue current.

:::

Select an operation to get started:

- **[Add datasets](/catalogue-managers-guide/manage-datasets-manually/add)** - Create new dataset records with metadata
- **[Add data resources](/catalogue-managers-guide/manage-datasets-manually/add-resources)** - Add files or links to your datasets
- **[Delete datasets](/catalogue-managers-guide/manage-datasets-manually/delete)** - Permanently remove datasets from your catalogue
- **[Manage dataset groups](/catalogue-managers-guide/manage-datasets-manually/manage-groups)** - Organise datasets into discoverable groups
