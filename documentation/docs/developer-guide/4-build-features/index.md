This section provides step-by-step guides for building features, organised by what you want to achieve, regardless of which component you're working on.

## Available guides

### UI development
- **[Add a new UI component](./add-a-new-ui-component/)**: Create React components for the frontend

### API development
- **[Add a new API endpoint](./add-a-new-api-endpoint/)**: Extend backend services with new REST endpoints

### Metadata and schema
- **[Add metadata fields](./add-metadata-fields/)**: Add new fields across CKAN, Solr, and services
- **[Work with CKAN schemas](./work-with-ckan-schemas/)**: Create and modify dataset schemas

### Data integration
- **[Develop harvesters](./develop-harvesters/)**: Build custom harvesters for data sources
- **[Integrate external APIs](./integrate-external-apis/)**: Connect to third-party services like REMS

### Data management
- **[Manage database migrations](./manage-database-migrations/)**: Create and apply database schema changes

### Security
- **[Implement authentication](./implement-authentication/)**: Work with OIDC flows and Keycloak integration
