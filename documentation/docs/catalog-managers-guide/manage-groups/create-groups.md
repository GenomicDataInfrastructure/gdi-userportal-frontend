# Create groups

Use **groups** to create and manage collections of datasets based on any creteria such as a particular project, team, or theme. Groups provide a way to:

Groups improve dataset discoverability by helping users find related datasets more easily. Here are some ways to use groups:

- **Group datasets by project:** Organise datasets related to a specific research project.
- **Group datasets by team:** Collect datasets managed by particular teams or departments.
- **Group datasets by theme:** Organise datasets around specific topics or research areas.

## Create a group

1. Select **Groups** from the main menu.

2. Select **Add Group**. 

3. Fill in the following fields:

   - **Name:** Enter a descriptive name for the group. The URL generates automatically based on the name. To change it, select *Edit* next to the generated URL. 
   - **Description:** (Optional) Enter a description or overview of the group.
   - **Image URL:** (Optional) Add an image URL to visually represent the group. This image appears as a thumbnail whenever the group is listed.

3. Select **Create Group**. After the group is successfully created, the list of groups appears with the newly created group.
